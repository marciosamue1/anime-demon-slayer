# 🔥 Demon Slayer Project

Este é um projeto inspirado no anime Demon Slayer, desenvolvido para praticar HTML, CSS e estruturação de páginas web.

---

## 🚀 Funcionalidades

- Exibição de conteúdo temático
- Layout estilizado
- Estrutura organizada

---

## 🛠️ Tecnologias utilizadas

- HTML
- CSS

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- Estruturação com HTML
- Estilização com CSS
- Organização de layout

---

## 📂 Como executar

Abra o arquivo `index.html` no navegador.

---

## 📚 Aprendizados

- Estrutura de páginas web
- Uso de CSS
- Organização visual
